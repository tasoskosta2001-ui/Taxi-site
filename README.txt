Τα αρχεία σε αυτό το πακέτο:
- index.html     : Η στατική ιστοσελίδα κράτησης ταξί (σε Ελληνικά).
- server.py      : Μικρός Python server για τοπική δοκιμή (python 3).
- README.txt     : Αυτό το αρχείο.

Οδηγίες τοπικής εκτέλεσης:
1) Εξάγαγε όλα τα αρχεία σε έναν φάκελο στον υπολογιστή ή στο iPad/Phone (Pythonista).
2) Άνοιξε τερματικό / Pythonista και τρέξε:
   python server.py
3) Άνοιξε browser και πήγαινε στο:
   http://localhost:8000/index.html

Άλλες επιλογές (χωρίς Python):
- Απλά ανέβασε το index.html σε GitHub Pages, Netlify ή Vercel (δωρεάν) για να είναι live στο internet.
- Αν θες, μπορώ να σε καθοδηγήσω βήμα-βήμα για GitHub Pages ή Netlify.
